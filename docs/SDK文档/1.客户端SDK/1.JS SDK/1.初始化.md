### `tcb.init`

### 参数说明

| 字段 | 类型   | 必填 | 说明 |
| :--: | :--: | :--: | -- |
| `env`  | `string` | 是   | 环境ID，前往腾讯云[云开发控制台](/2019-09-03-web-dev-guide-console/)获取。 |

### 示例代码

```javascript
// 初始化示例
const app = tcb.init({
  env: "texample-envid"
});
```
